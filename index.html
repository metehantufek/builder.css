What is Builder.css?

Builder.css is a CSS library made for websites that want to replicate an old DOS style. It relies heavily on the concept of semantic HTML.
Although the library on its own doesn't use any JavaScript code at all, being made only from HTML & CSS, there are small JavaScript modules present within this site that can enhance the aesthetic and feeling of the DOS style.
Builder.css is compatible with most monospace fonts.
